# 📚 Elevate Study

**Elevate Study** is a personal passion project built to help students stay organized, consistent, and productive. It brings together essential productivity tools and curated study resources in one simple platform.

This project focuses only on **productivity and study management**. It does **not include AI note-taking, summarization, or content generation features**.

---

## 🚀 Features

### 🧠 Productivity Tools
- To-do list manager
- Task tracking system
- Daily streak tracker
- Study goal setting
- Progress tracking system

### 🛠️ Student Tools Library
- Curated list of free and paid tools for students
- Useful AI tools (for productivity support only)
- Categorized resources (study, coding, design, research, etc.)
- External links to helpful platforms

### 📊 Study System
- Daily planning structure
- Weekly study tracking
- Habit-building through streaks
- Goal completion monitoring

---

## 🎯 Purpose

The goal of Elevate Study is to:

- Help students stay consistent with studying  
- Improve productivity habits  
- Reduce procrastination  
- Organize tasks and study goals in one place  
- Provide access to useful learning tools

## 📦 Installation

```bash
git clone https://github.com/your-username/elevate-study.git
cd elevate-study
npm install
npm run dev
```

## 💻 Run Locally

Start the development server:
```bash

npm run dev
```

Then open:
```bash

http://localhost:3000
```
